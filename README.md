**Learnix** is a **single-page, front-end gamified learning system** that combines **adaptive exams, XP-based progression, daily challenges, and unlockable themes** into one immersive student dashboard. Built entirely with **vanilla HTML5, CSS3, and JavaScript**, it delivers a futuristic glassmorphism UI with zero backend dependency.

---

## 🚀 Key Features

- **🔐 Student Authentication** — Lightweight login system with demo credential support
- **📚 Multi-Subject Exam Engine** — 10 academic domains (Web Dev, Cybersecurity, Data Structures, Networks, Math, and more)
- **🎯 Adaptive Difficulty Levels** — Easy / Medium / Hard tiers with timed, auto-generated question sets
- **⏱️ Real-Time Exam Timer** — Countdown-based session control with auto-submit on timeout
- **📊 Detailed Results & Analytics** — Score circle visualization, correct/wrong breakdown, mistake review
- **🕓 Persistent Exam History** — LocalStorage-backed history log with average score tracking
- **🔥 Daily Challenge System** — Date-seeded rotating tasks with XP rewards and completion tracking
- **⭐ XP & Progress System** — Gamified point accumulation with level-progress bar
- **🎨 Unlockable Theme Engine** — XP-gated visual themes (Blue, Pink, Cyber, Galaxy, Ultra)
- **📱 Fully Responsive Design** — Optimized layout across desktop, tablet, and mobile breakpoints
- **✨ Modern Glassmorphism UI** — Neon gradients, blur effects, and smooth animated transitions
